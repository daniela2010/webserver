Custom Multi-Threaded Web Server

A high-performance HTTP server built entirely from scratch for our Computer Networks course.

Explore Project Specifications

Core Server Architecture

Our custom-built web server is capable of handling incoming TCP socket requests directly at the Transport Layer. By reading byte streams from client requests, parsing HTTP headers, and validating requested resources, it processes web traffic efficiently.

This project focuses on understanding how real web servers operate internally by implementing sockets, HTTP request parsing, resource resolution, MIME type detection, and multi-threaded client handling from scratch.

Supported Content Types: text/html, text/css, image/jpeg, image/png
